    Blender за 1 час
    1 час · Blender · моделлинг
    Blender — мощный бесплатный инструмент для 3D-моделирования. За один час вы познакомитесь с интерфейсом, создадите первый объект, настроите материал и сделаете простой рендер.
    01
    Интерфейс Blender
    Blender состоит из редакторов: 3D Viewport (рабочая область), Outliner (список объектов справа), Properties (свойства справа снизу) и Timeline (анимация снизу). Каждый редактор можно изменять и переключать под задачи.
    02
    Первый объект
    Нажмите Shift+A, чтобы открыть меню добавления объектов. Выберите Mesh → Cube. Объект появится в центре сцены. Нажмите Tab, чтобы войти в Edit Mode и редактировать геометрию.
    03
    Материалы и цвет
    Откройте вкладку Material Properties (иконка сферы в Properties). Нажмите «New», чтобы создать материал. В разделе Base Color выберите любой цвет. Переключитесь на Rendered Viewport Shading (кнопка в правом верхнем углу viewport), чтобы видеть результат.
    04
    Рендер
    Нажмите F12, чтобы запустить рендер. Blender использует движок Cycles (реалистичный) или EEVEE (быстрый). Для начала используйте EEVEE — результат появится за секунды. Сохраните изображение через Image → Save As.
    Совет: в Blender все горячие клавиши работают там, где находится курсор мыши. Убедитесь, что мышь находится во viewport, прежде чем нажимать G (move), R (rotate), S (scale).
