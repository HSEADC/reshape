    Маршмеллоу заяц в Blender
    40 минут · Blender · моделлинг
    Создадим милого мягкого зайца из простых форм в Blender. Этот туториал идеален для начинающих — мы используем только базовые операции: вытягивание, масштабирование и модификатор Subdivision Surface для сглаживания.
    01
    Тело зайца
    Начните с UV Sphere (Shift+A → Mesh → UV Sphere). В Edit Mode сплющьте его снизу (выделите нижние вертексы, S → Z → 0.7). Добавьте модификатор Subdivision Surface с уровнем 2 — форма станет мягкой и округлой.
    02
    Голова и уши
    Добавьте ещё одну сферу поменьше для головы. Для ушей создайте вытянутые сферы: добавьте Sphere, в Edit Mode выделите всё (A), нажмите S → Z → 3, чтобы вытянуть. Разместите уши сверху головы и слегка наклоните их.
    03
    Материал маршмеллоу
    Создайте материал с Base Color белого или нежно-розового цвета. Увеличьте Roughness до 0.9 — маршмеллоу матовый. Добавьте немного Subsurface Scattering (Subsurface = 0.1, цвет — тёплый розовый) для эффекта мягкой светопроницаемости.
    04
    Финальный рендер
    Добавьте Area Light над моделью. Установите белый фон (World → Background → белый). Переключитесь на движок EEVEE для быстрого рендера. Нажмите F12 и сохраните результат.
    Совет: используйте Proportional Editing (O) при работе с формой тела — это позволяет плавно деформировать группы вертексов, создавая мягкие органические формы.
