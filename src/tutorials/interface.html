    Интерфейс по-быстрому
    10 минут · Cinema 4D · моделлинг
    Давайте быстро познакомимся с основными элементами интерфейса Cinema 4D. Этот туториал поможет вам ориентироваться в программе с первых минут работы — без лишней теории, только самое нужное.
    01
    Viewport — рабочая область
    Центральная часть экрана — это viewport. Здесь вы видите вашу сцену. Можно переключаться между видами: перспектива, спереди, сбоку, сверху. Для этого нажмите на иконку камеры в верхнем левом углу viewport.
    02
    Object Manager — список объектов
    Справа сверху находится Object Manager — здесь перечислены все объекты сцены в виде иерархии. Кликайте на объект, чтобы выделить его. Перетаскивайте объекты, чтобы создавать группы.
    03
    Attribute Manager — свойства
    Справа снизу — Attribute Manager. Здесь вы настраиваете параметры выбранного объекта: размер, положение, поворот, цвет и другие свойства. Все изменения видны сразу во viewport.
    04
    Панель инструментов
    Слева — основные инструменты: перемещение (E), масштаб (T), поворот (R). Верхняя панель содержит меню для создания примитивов, источников света, камер и спецэффектов.
    Совет: зажмите Alt и тяните мышью во viewport, чтобы вращать камеру. Alt + правая кнопка мыши — зум. Alt + средняя кнопка — перемещение камеры.
